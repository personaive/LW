# Задание 1

## Задача 11

### Текст задачи

Сформировать список из длин вводимых строк

### Алгоритм решения

1. Ввести количество строк с клавиатуры.

2. Проверить, что введено число и оно больше или равно нулю.

3. Если проверка не пройдена, вывести сообщение об ошибке и завершить программу или перезапросить ввод.

4. Создать список, считывая каждую строку от пользователя.

5. Для каждой строки вычислить её длину.

6. Сформировать список длин строк.

7. Вывести список длин строк на экран.

# Задание 2

## Задача 11

### Текст задачи

Найти сумму чётных цифр натурального числа

### Алгоритм решения

1. Ввести натуральное число с клавиатуры.

2. Преобразовать число в строку, чтобы работать с каждой цифрой отдельно.

3. Разбить строку на список символов (каждая цифра - отдельный символ).

4. Преобразовать каждый символ в число.

5. Проверить каждое число, является ли оно чётным.

6. Если цифра чётная, включить её в сумму.

7. Суммировать все чётные цифры.

8. Вывести результат на экран.

# Задание 3

## Задача 1

### Текст задачи

Опишите функции, осуществляющие основные математические операции над комплексными
числами (сложение, вычитание, умножение, деление, возведение в степень). Использовать
готовый класс Complex запрещено.

### Алгоритм решения

1. Ввести действительную часть первого числа с клавиатуры.

2. Ввести мнимую часть первого числа с клавиатуры.

3. Ввести действительную часть второго числа с клавиатуры.

4. Ввести мнимую часть второго числа с клавиатуры.

5. Вывести меню доступных операций.

6. Прочитать выбранную операцию.

7. Если выбрана операция «возведение в степень», запросить число степени.

8. Выполнить выбранную операцию над комплексными числами.

9. Вывести результат на экран.
