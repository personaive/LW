# Задание 1

## Задача 11

### Текст задачи

Сформировать список из длин вводимых строк

### Алгоритм решения

1\. Попросить пользователя ввести количество строк n

2\. Создать рекурсивную функцию readLengths(n)

3\. Если n <= 0, функция возвращает пустой список \[]

4\. Берётся длина строки и добавляется в начало списка с помощью ::

5\. Рекурсивно вызывается readLengths(n-1) для оставшихся строк

6\. В main вызвать readLengths(n) и сохранить результат в lengths

7\. Вывести список lengths на экран

# Задание 2

## Задача 11

### Текст задачи

Найти сумму чётных цифр натурального числа

### Алгоритм решения

1\. Попросить пользователя ввести натуральное число number

2\. Создать рекурсивную функцию sumEvenDigits(n)

3\. Если n = 0, функция возвращает 0

4\. Иначе берётся последняя цифра числа: digit = n % 10

5\. Вычисляется оставшееся число: rest = n / 10

6\. Если digit чётная, возвращаем digit + sumEvenDigits(rest)

7\. Если digit нечётная, возвращаем sumEvenDigits(rest)

8\. В main вызвать sumEvenDigits(number) и сохранить результат в result

9\. Вывести result на экран

# Задание 3

## Задача 1

### Текст задачи

Опишите функции, осуществляющие основные математические операции над комплексными
числами (сложение, вычитание, умножение, деление, возведение в степень). Использовать
готовый класс Complex запрещено.

### Алгоритм решения

1\. Попросить пользователя ввести действительную часть первого числа aRe

2\. Попросить пользователя ввести мнимую часть первого числа aIm

3\. Попросить пользователя ввести действительную часть второго числа bRe

4\. Попросить пользователя ввести мнимую часть второго числа bIm

5\. Вывести меню операций: 1 - сложение, 2 - вычитание, 3 - умножение, 4 - деление, 5 - возведение в степень

6\. Считать выбор пользователя choice

7\. Создать функцию add(a,b) для сложения: (aRe+bRe, aIm+bIm)

8\. Создать функцию subtract(a,b) для вычитания: (aRe-bRe, aIm-bIm)

9\. Создать функцию multiply(a,b) для умножения: (aRebRe - aImbIm, aRebIm + aImbRe)

10\. Создать функцию divide(a,b) для деления: ((aRebRe + aImbIm)/(bRe^2+bIm^2), (aImbRe - aRebIm)/(bRe^2+bIm^2))

11\. Создать рекурсивную функцию power(a,n) для возведения в степень: если n=0, вернуть (1,0); если n=1, вернуть (aRe,aIm); иначе рекурсивно вызвать power(a,n-1) и умножить результат на (aRe,aIm)

12\. В main вызвать функцию, соответствующую choice, и сохранить результат (resRe,resIm)

13\. Вывести результат в формате resRe + resIm i или resRe - abs(resIm) i, если мнимая часть отрицательна
